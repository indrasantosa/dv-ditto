## How to use DV SIN Design System
Only use this package if you're using following framework:
1. Create React App
2. Next.js (Coming soon)

```shell
# Install dv-ditto on global scope
npm install -g dv-ditto

# On the root of the project, run the following command
dv-ditto

# Install dependencies
yarn install

# Include tailwindcss styles into the main project css file
# @tailwind base;
# @tailwind components;
# @tailwind utilities;

# Run your project normally
yarn start
```
